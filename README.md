# Banco de Talentos Pro

Sistema de gerenciamento de talentos com dashboard administrativo e formulário público de candidatura.

## Funcionalidades

- 📋 Dashboard administrativo com listagem de candidatos
- 📝 Formulário público de candidatura
- 📊 Exportação de dados (XLS/PDF)
- 🔍 Filtros e busca avançada
- 🗄️ Integração com Supabase (PostgreSQL)

## Tecnologias

- HTML5 / CSS3 / JavaScript
- Supabase (Backend as a Service)
- PostgreSQL

## Configuração

1. Crie um projeto no [Supabase](https://supabase.com)
2. Execute o arquivo `schema.sql` no SQL Editor do Supabase
3. Atualize as credenciais do Supabase no arquivo `banco-talentos-pro.html`
4. Abra o arquivo `banco-talentos-pro.html` no navegador

## Estrutura

```
├── banco-talentos-pro.html  # Aplicação principal (dashboard + formulário)
├── schema.sql               # Schema do banco de dados (PostgreSQL/Supabase)
└── README.md                # Documentação
```
